2019년 7월~2021년 1월 (1년 7개월) / IT 대기업 기존 시스템 유지보수
B2C
Vue.js

프로젝트 개요

푸드 테이크아웃 앱의 안정적인 운영과 기능 개선을 목표로 기존 서비스의 유지보수 및 기능 추가를 담당했습니다. 사용자 경험을 해치지 않도록 프론트엔드의 버그 수정, 기능 확장, 멀티 디바이스 대응 최적화를 지속적으로 수행하며, 서비스 품질과 운영 체계를 강화해 비즈니스 성장의 기반을 지원한 프로젝트입니다.

담당 단계

  • 유지보수

체계 및 역할

  • 전체 60명, 팀 15명
  • 프론트엔드 개발자

업무 내용

  • 프론트엔드 버그 수정 및 소규무 기능 개선
  • 기존 기능의 확장 및 개편을 위한 추가 개발
  • 스마트폰, 태블릿, PC 등 멀티 디바이스 대응을 위한 UI 조정

실적 및 활동

  • 담당한 모든 버그 티켓을 꾸준히 해결하여 안정적 운영 기여
  • UI 호환성을 개선하여 다양한 기기에서의 사용자 경험 향상 달성

기술 스택

  • 프론트엔드: Vue.js, JavaScript
  • 백엔드: Java, SpringBoot
  • DB: MySQL
  • 인프라: 온프레미스, 리눅스
