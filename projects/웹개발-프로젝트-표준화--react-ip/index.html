웹개발 프로젝트 표준화 : React IP 👨‍💻

2024년 9월~2025년 5월(9개월) / 사내 표준화 활동
React
TypeScript

프로젝트 개요

사내 React 개발의 효율화 및 고품질화를 위해 공통 라이브러리(IP)의 유지 운영 및 도입 지원을 진행하였습니다. 각 팀의 도입 지원을 통해 개발 표준의 정착과 개발 리소스 절감을 추진하였으며, React 고유의 이슈를 정리하고 재사용 가능한 컴포넌트 정비 및 버전 관리 체계를 구축하였습니다.

담당 단계

  • 유지보수 및 운영

체계 및 역할

  • 전체 7명, 팀 7명
  • 프로젝트 리더

업무 내용

  • React IP 도입 지원 (도입 방법 설명 및 활용 제안)
  • 공통 부품의 정기적인 버전업
  • 해외 벤더와의 영어로 과제 공유 및 업무 조정

실적 및 활동

  • 모든 도입 대상 프로젝트에 React IP의 개요와 도입 방법을 설명하고 지원하여, 원활한 적용을 유도함. 그 결과 표준화 비용을 50% 이상 절감
  • 3개월마다 React IP의 정기적인 버전 업그레이드를 통해 최신 상태 유지
  • 각 프로젝트 도입처에서 정기적으로 React IP 관련 피드백을 수집하고, 요구사항을 반영한 부품 수정 및 기능 추가를 지속적으로 추진
  • 해외 벤더 및 본사와의 가교 역할을 맡아, 영어로 이슈를 공유하고 원활한 커뮤니케이션을 통해 업무 효율화에 기여

기술 스택

  • 프론트엔드: React, TypeScript
  • 인프라: AWS, Linux
