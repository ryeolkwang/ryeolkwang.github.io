フードテイクアウトサービスの維持保守 🥡

2019年7月～2021年1月（1年7ヶ月）　／　大手IT会社向け既存システムの維持保守
B2C
Vue.js

プロジェクト概要

フードテイクアウトアプリの安定運用と改善を目的に、既存サービスの保守・機能改善を担当。ユーザー体験を損なわないようフロントエンドの不具合修正や機能追加、マルチデバイス対応の最適化を継続的に実施。サービス品質と運用体制の強化により、ビジネス成長を下支えする役割を果たしました。

担当フェーズ

  • 維持保守

体制・役割

  • 全体60名、チーム15名
  • フロントエンド開発者

業務内容

  • フロントエンドのバグ修正と軽微な機能改善
  • 既存機能の拡張・改修に向けた追加開発
  • スマホ・タブレット・PCなどマルチデバイス対応のUI調整

実績・取り組み

  • 担当したすべてのバグチケットを着実に解消し、安定稼働に貢献
  • UI互換性の向上により、幅広い端末での利用体験を改善

技術スタック

  • フロントエンド: Vue.js, JavaScript
  • バックエンド: Java, SpringBoot
  • DB: MySQL
  • インフラ: オンプレミス, Linux
